## About

NovaTech Solutions is a career guidance platform designed to support students and aspiring professionals in exploring technology careers.

The platform analyzes users' interests and skills, recommends suitable career paths, and provides structured learning roadmaps with carefully selected certifications. It bridges the gap between education and industry by helping learners make informed career decisions.

## Installation

Clone the repository

```bash
git clone https://github.com/USERNAME/NovaTech.git
```

Go to the project

```bash
cd NovaTech
```

### Install Backend

```bash
cd backend
npm install
```

Run the backend

```bash
npm start
```

### Install Frontend

```bash
cd frontend
npm install
npm run dev
```
